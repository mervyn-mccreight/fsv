Required Softwares

BOBJ is 100% pure Java application, and it runs on any platform in which Java2 runtime system is installed. no more software is required.

What to download?

BOBJ has being packaged in two different ways: as a unix compressed archive (bobj0.9.tar.gz), and as a zip file (bobj0.9.zip). You can download either one according to uncompress tools on your machine.

How to Run BOBJ?

  1. if you save all files in a directory $bobj, you can run bobj by the following command under that directory:
    in Unix
           java -jar $bobj/bobj.jar
    in Windows
           java -jar $bobj\bobj.jar
  2. if you want to run BOBJ from any directory, you can save this command as a shell script or .bat file, and add this script to your PATH. For convenience, two scrpts are included in the bin directory, you can edit them and change $bobj to the directory where you install BOBJ.

How to Execute Examples

After you start BOBJ, you will see a prompt "BOBJ>" from BOBJ, then you can input BOBJ commands to do your work. Here is a typical procedure of running BOBJ specification: [cat:/tmp/bobj] java -cp /tmp/bobj.jar bobj.BOBJ --- Welcome to BOBJ --- /|||||||||||||||||\ BOBJ version 0.9.127 built: Tue Oct 30 16:06:59 PST 2001 University of California, San Diego Thu Nov 01 17:00:27 PST 2001 BOBJ> cd demos/view BOBJ> ls assoc.bob map.bob pv.bob pview.bob tree.bob BOBJ> in map ========================================== obj LIST ========================================== th FUN ========================================== obj MAP ========================================== obj DOUBLE ========================================== make MAP-double BOBJ>

BOBJ Commands

Following 4 commands provide you the interface with operating system: ls -- list all files in the current directory pwd -- show the current directory in <filename> -- load specified bob file and execute it q -- quit BOBJ More commands about specification can be found from examples.
